### 条件执行  

- 逻辑与&& 
`command1 && command2` 只有当command1执行成功，才会执行command2  
 

```shell
rm somefile && echo "file deleted"  

read num1
read num2
if [ $num1 -le 100 -a $num1 -le $num2 ]
then 
	echo "num1 < num2 < 100"i
fi


read num3
read num4
if [ $num3 > $num4 ] && [ $num3 < 100 ]
then
	echo "num3 < num4 < 100"
fi
```
