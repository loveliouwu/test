- 无论是自定义的属性或者类自带的属性，都会有响应的属性值变化的槽函数  
- 一般属性的命名规则，id属性的值，首字母必须是小写字母或下划线，且不能包含字母数字下划线以外的字符，一般第一个单词消息第二个单词首字母大写如testColor
```
Item{
    property int count:10  //自定义的属性
    width : 100 //类自带的属性


    //则系统默认生成属性改变的槽函数
    onWidthChanged:{
        console.log("width change");
    }

    onCountChanged:{
        console.log("count change");
    }


}
```
